# 📖 Overview

The **Internship Recommendation System** is a Spring Boot web application developed to simplify the internship search and application process for students.

Students can browse internships, view recommendations based on their skills, upload resumes, and submit internship applications through a user-friendly interface.

---

# ✨ Features

- 🎯 Skill-based internship recommendations
- 🔍 Browse available internships
- 📝 Online internship application form
- 📄 Resume upload functionality
- 💾 H2 Database integration
- ⚡ Fast Spring Boot backend
- 🎨 Responsive UI using HTML, CSS & Thymeleaf
- 📱 Mobile-friendly interface

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Jhanani85/internshiprecommendation.git
```

---

## Open Project

Open using

- IntelliJ IDEA
- Eclipse
- VS Code

---

## Run Application

```bash
mvn spring-boot:run
```

or run

```
InternshipRecommendationApplication.java
```
